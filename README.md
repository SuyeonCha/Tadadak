# Tadadak

<b>개발시기</b> : 2018.04 - 2018.07 (4人 프로젝트)  <br>
<b>개발 언어 및 툴</b> : C# / Visual Studio
<br>
<br> 
<h3>프로그램 내용 </h3>
타자 연습과 게임을 할 수 있는 프로그램입니다. 짧은 글과 긴 글 그리고 산성비 게임으로 타자 연습을 하며 실력을 키우고, 누구나 쉽고 재밌게 즐길 수 있기 바라며 개발했습니다.

<h3>주요기능 </h3>
짧은 글 연습, 긴글 연습, 타자 게임
